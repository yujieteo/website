This is the personal website of Yu Jie Teo.

I am broadly interested in mathematics, computing, physics, and engineering. This website is a collection of notes, references, and technical material that I want to revisit and develop over time.

  1. Notes in mathematics. The objective of these notes is to have my mathematical thinking clearly written and organized in a form that can be revisited later. My LaTeX notes are now here.
  2. General annotated bibliography. A collection of books, papers, and references across different fields with short notes and observations. This is meant to be a more general dump.
  3. General exercises and problems. Exercises and problem sets intended to accompany the material in the general reading list.
  4. Selected reading list with exercises. The objective is to focus readings into concrete computations and problems to accelerate future understanding. Eventually I want to port most of my mathematics reading list over.
  5. General mathematics annotated bibliography. A larger archive of accumulated mathematics references with sparse notes for later searching and exploration. This is very big and I am trying to trim it or make it obsolete in contrast ot math exercises.
  6. Core systems programming and hardware references.
    • Python Documentation. The official reference for the Python language and standard library. Python was the first language I truly learnt and can go back to.
    • C# Documentation. Official documentation for the C# language and the .NET ecosystem. I use this for work.
    • Intel x86/x86-64 Software Developer Manuals. The primary reference for x86 processors, instruction sets, memory models, and system programming.
    • Git Documentation. Official documentation for Git version control.
    • LLVM Documentation. Reference material for compiler infrastructure, optimization, and code generation.
    • QEMU Documentation. Documentation for machine emulation, virtualization, and systems experimentation. Eventually I will want to do virtualisation.
  7. Computer architecture and hardware-accelerated computing references.
    • NVIDIA CUDA Toolkit Documentation. Documentation for GPU programming, parallel algorithms, and accelerator-based computing.
    • Competitive Programming Algorithms. A practical collection of algorithms and data structures with implementations and explanations. This is an easy reference to consult.
    • A Modern Primer on Processing in Memory (PIM). An overview of architectures that move computation closer to memory in order to reduce data movement costs.
  8. Mathematics, mathematical physics, and research references. These references support long-term study, ranging from introductory lecture notes to advanced research literature.
    • David Tong's Lecture Notes. Introductions to mathematical physics, quantum field theory, statistical physics, relativity, and related topics. I will use this reference.
    • J.S. Milne's Mathematical Notes. Graduate-level notes covering algebraic geometry, algebraic number theory, class field theory, arithmetic geometry, and related fields. Milne's notes are still the gold standard for a lot of topics in terms of exposition, particularly on anything related to Tannakian duality.
    • arXiv Mathematics. A source for current mathematical research papers and preprints. I need to basically try and pick out good stuff from here.
    • Stacks Project. A comprehensive reference for algebraic geometry, commutative algebra, and foundational topics. I remember skimming through the entirety Chapter 10 on commutative algebra.
    • Kerodon. Notes on higher category theory, homotopy theory, and modern foundations.
    • 1Lab. Formalized and computational approaches to category theory and higher mathematics. This is a handy reference to see how a proof of something would look like when formalised.
    • nLab. An interconnected encyclopedia covering category theory, topology, mathematical physics, and higher structures.
    • pi-base, topology. A database of examples and counterexamples in topology.
    • Friedl's Topology Notes. An extensive reference covering large portions of modern topology.
    • LMFDB. A database for exploring elliptic curves, modular forms, L-functions, and related objects in number theory. It is fun exploring this.
    • zbMATH Open. A searchable database of mathematical literature and reviews.
    • Cambridge Mathematics Notes. A collection of university mathematics lecture notes spanning many subjects.
  9. Computational and experimental mathematics. I am now increasingly keen on this topic.
    • The On-Line Encyclopedia of Integer Sequences (OEIS). A searchable database of integer sequences that is often useful when identifying patterns arising from calculations.
    • Wolfram Language & System Documentation Center. Wolfram Language is still one of the most powerful mathematical computer algebra systems with sane defaults for notebooks unlike Jupyter.
    • SageMath Documentation. I want to learn Sage eventually as a CAS.
    • Kaggle. There is probably some data set from here that can be used to do data science.
    • Avigad - Theorem Proving in Lean 4. An introduction to formal mathematics and theorem proving with Lean.
  10. Engineering references, with an emphasis on structures, simulation, numerical methods, and aerospace applications.
    • PATRAN Documentation. I use this preprocessor for work.
    • ANSYS APDL. This is the first finite element software that I learnt deeply with mentorship at internship.
