This is the personal website of Yu Jie Teo.

I am broadly interested in mathematics, computing, physics, and engineering. This website is a collection of notes, references, and technical material that I want to revisit and develop over time.

  1. Notes in mathematics. The objective of these notes is to have my mathematical thinking clearly written and organized in a form that can be revisited later.
  2. General annotated bibliography. A collection of books, papers, and references across different fields with short notes and observations.
  3. General exercises and problems. Exercises and problem sets intended to accompany the material in the general reading list.
  4. Selected reading list with exercises. The objective is to focus readings into concrete computations and problems to accelerate future understanding. Eventually I want to port most of my mathematics reading list over.
  5. General mathematics annotated bibliography. A larger archive of accumulated mathematics references with sparse notes for later searching and exploration.
  6. Core systems programming and hardware references. These are foundational references that I repeatedly return to while studying programming languages, operating systems, compilers, and computer architecture.
    • Python Documentation. The official reference for the Python language and standard library.
    • C# Documentation. Official documentation for the C# language and the .NET ecosystem.
    • Intel x86/x86-64 Software Developer Manuals. The primary reference for x86 processors, instruction sets, memory models, and system programming.
    • POSIX / Single UNIX Specification. The standard reference for portable Unix-like operating system interfaces.
    • Linux man pages. Reference documentation for Linux system calls, libraries, and utilities.
    • Git Documentation. Official documentation for Git version control.
    • MIT Operating Systems Engineering. A practical operating systems course emphasizing kernels, virtualization, and systems design.
    • Linux Kernel Documentation. Documentation covering Linux kernel internals, subsystems, and development.
    • LLVM Documentation. Reference material for compiler infrastructure, optimization, and code generation.
    • GCC Documentation. Documentation for the GNU Compiler Collection and its toolchain.
    • QEMU Documentation. Documentation for machine emulation, virtualization, and systems experimentation.
  7. Computer architecture and hardware-accelerated computing references.
    • NVIDIA CUDA Toolkit Documentation. Documentation for GPU programming, parallel algorithms, and accelerator-based computing.
    • CP Algorithms. A practical collection of algorithms and data structures with implementations and explanations.
    • A Modern Primer on Processing in Memory (PIM). An overview of architectures that move computation closer to memory in order to reduce data movement costs.
  8. Mathematics, mathematical physics, and research references. These references support long-term study, ranging from introductory lecture notes to advanced research literature.
    • David Tong's Lecture Notes. Introductions to mathematical physics, quantum field theory, statistical physics, relativity, and related topics.
    • J.S. Milne's Mathematical Notes. Graduate-level notes covering algebraic geometry, algebraic number theory, class field theory, arithmetic geometry, and related fields.
    • arXiv Mathematics. A source for current mathematical research papers and preprints.
    • Stacks Project. A comprehensive reference for algebraic geometry, commutative algebra, and foundational topics.
    • Kerodon. Notes on higher category theory, homotopy theory, and modern foundations.
    • 1Lab. Formalized and computational approaches to category theory and higher mathematics.
    • nLab. An interconnected encyclopedia covering category theory, topology, mathematical physics, and higher structures.
    • pi-base, topology. A database of examples and counterexamples in topology.
    • Friedl's Topology Notes. An extensive reference covering large portions of modern topology.
    • LMFDB. A database for exploring elliptic curves, modular forms, L-functions, and related objects in number theory.
    • zbMATH Open. A searchable database of mathematical literature and reviews.
    • MathOverflow. A research-level discussion forum used for mathematical questions, references, and expert insight.
    • Wolfram MathWorld. A quick reference encyclopedia for mathematical definitions and examples.
    • Cambridge Mathematics Notes. A collection of university mathematics lecture notes spanning many subjects.
  9. Computational and experimental mathematics. Using computation to discover patterns, test conjectures, verify calculations, and build mathematical intuition.
    • The On-Line Encyclopedia of Integer Sequences (OEIS). A searchable database of integer sequences that is often useful when identifying patterns arising from calculations.
    • Wolfram Language & System Documentation Center. Documentation covering symbolic computation, visualization, numerical methods, and scientific programming.
    • SageMath Documentation. Reference documentation for SageMath and its mathematical software ecosystem.
    • Kaggle. A large collection of datasets and computational notebooks useful for experimentation, machine learning, and data analysis.
    • Magma Computational Algebra System. A powerful system for computation in algebra, geometry, combinatorics, and number theory.
    • SageMath. Open-source mathematical software integrating many specialized computational tools.
    • SymPy. A Python library for symbolic mathematics and exact computation.
    • GAP Computer Algebra System. Software focused on computational group theory and discrete algebra.
    • Avigad - Theorem Proving in Lean 4. An introduction to formal mathematics and theorem proving with Lean.
  10. Engineering references, with an emphasis on structures, simulation, numerical methods, and aerospace applications.
    • NASA Technical Reports Server. A large archive of engineering reports, technical memoranda, design studies, and aerospace research publications.
    • MMPDS-08. The Metallic Materials Properties Development and Standardization handbook, a standard aerospace reference for engineering material properties.
    • PATRAN Documentation. Documentation for finite element preprocessing, meshing, and analysis workflows.
    • ANSYS APDL. Reference documentation for finite element analysis and engineering simulation.
